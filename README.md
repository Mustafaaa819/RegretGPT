# ⚰️ RegretGPT: The Cemetery of Life Choices

RegretGPT is a brutally honest AI agent that analyzes your bad decisions, challenges your mental models, and writes a dark-comedy funeral eulogy for your mistakes.

## 🧠 Features
- **Decision Autopsy:** Brutal logic-based critique of your choices.
- **Regret Scoring:** A 0-10 scale of how much you should regret your life.
- **Dark Comedy Eulogies:** Poetic funeral services for your "dead" decisions.
- **The Graveyard:** A persistent memory of every mistake you've ever buried.

## 🛠️ Tech Stack
- **Language:** Python
- **LLM:** Anthropic Claude 3.5 Sonnet
- **Memory:** JSON-based persistence

## 🚀 Setup
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Add your `ANTHROPIC_API_KEY` to a `.env` file.
4. Run `python main.py` and start burying your regrets.
